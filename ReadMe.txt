This repository contains SAS PROC FCMP implementations of some common data linking functions as contained in the
ICARUS data linking system.

C implementations have not been included.

To use these functions in SAS, you will need to acquaint yourself with the SAS documentation surrounding SAS
and the access of user based functions in PROC FCMP.

These functions are given as is, with absolutely no guaruntee or support.

Feedback, queries, and bug reports are welcome.
